# 📚 DSA — Data Structures & Algorithms

> My complete DSA learning journey using **C++**, following the structured curriculum of **[CodeHelp by Love Babbar](https://www.codehelp.in/)**.

---

## 🧑‍💻 About This Repository

This repository contains all my practice code, notes, and implementations from the **CodeHelp DSA course by Love Babbar** — one of the most well-known and beginner-friendly DSA courses in India. The course follows a week-by-week structure covering all major topics from fundamentals to advanced data structures.

Every folder here corresponds to a week of the course. The code is written in **C++**.

---

## 📂 Repository Structure

| Week | Topic |
|------|-------|
| `lecture2patterns` | Pattern Problems (Loops warmup) |
| `lecture6` | Basic Programming Concepts |
| `Week-03 Array` | Arrays — traversal, searching, manipulation |
| `Week-04 Searching & Sorting` | Binary Search, Sorting algorithms |
| `Week-05 Char array & String` | Character arrays, String operations |
| `Week-06 Pointers` | Pointers, memory, references |
| `Week-07 Recursion & Backtracking` | Recursion tree, Backtracking problems |
| `Week-08 Divide & Conquer` | D&C paradigm, Time & Space Complexity |
| `Week-09 OOPs` | Object-Oriented Programming in C++ |
| `Week-10 Linked List` | Singly, Doubly, Circular Linked Lists |
| `Week-11 Stack` | Stack implementation & problems |
| `Week-12 Queue` | Queue, Deque, Circular Queue |
